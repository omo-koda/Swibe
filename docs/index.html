Language Primitives

Every Swibe primitive is a first-class citizen — no libraries required.

think

LLM reasoning with SHA-256 receipt sealing. Works with Ollama, Claude, or OpenRouter.

swarm

Multi-agent coordination. Parallel execution with circuit breaker and shared state.

ethics

Runtime ethics enforcement. harm-none, audit-trail, or full hermetic mode.

birth

Sovereign agent identity via BIPỌ̀N39. Ed25519 keypair at first breath.

heartbeat

Autonomous proactive scheduling. Sabbath-aware. No cron needed.

remember

Persistent memory across sessions. SHA-256 keyed vault storage.

evolve

Soul evolution. Agent grows with every completed breath.

budget

Resource controls. Token limits, time limits, backpressure.

chain

Sequential reasoning steps. Output of one feeds next.

  ethics { harm-none; audit-trail }

}
