## 🚀 Quick Commands

### Compile Swibe Code
```bash
cd swibe
node src/index.js compile program.swibe
```

### Run Web IDE
```bash
cd swibe/web-playground
npm install
node server.js
# Open http://localhost:3000
```

### Run Tests
```bash
node src/index.js test tests/*.swibe
```

### Interactive REPL
```bash
node src/index.js repl
```

### Generate Documentation
```bash
node src/index.js docs
```

### Format Code
```bash
node src/index.js format program.swibe
```

```swibe
}
```

### With AI
```swibe

  %% classify this text as positive or negative
}
```

### API Endpoint
```swibe
#[api(method="GET", path="/users")]
fn getUsers(id: i32) -> str {
  "User: " + id.str()
}
```

```swibe
  // Schema definition
```
